Installation Instructions

To use this theme, you need to install it in a WordPress site:

  1. Install WordPress on your server
  2. Zip this entire directory
  3. Go to WordPress Admin → Appearance → Themes → Add New → Upload Theme
  4. Upload the zip file and activate the theme

Quick Start with Local WordPress

If you want to test this theme locally, you can use tools like:

  • Local by Flywheel - Easy WordPress local development
  • MAMP - Local server environment for Mac
  • Manual WordPress Installation

Using Docker (recommended)

This theme includes a docker-compose.yml file for easy setup:

cd /path/to/parent/directory
docker-compose up -d
# Visit http://localhost:8000 to install WordPress
# Then install the theme from the WordPress admin

This will create a complete WordPress environment with the theme already installed.
